Using the Application

Search for Error Codes

Enter an error code in the search box (e.g., E001, A020, W100)
Click "Search" or press Enter
View the error description and recommended remedies

Browse Available Codes

Scroll down to see all available error codes displayed as cards. Click any card to search for that code.

Search Tips

  • Search is case-insensitive (E001 and e001 both work)
  • You can search by code, title, or description
  • Try partial searches like "motor" to find related errors
  • Use category names like "Error", "Alert", "Warning" to filter

Adding Error Codes

To add error codes from your robot maintenance manual:

Step 1: Open data.js

Use any text editor (Notepad++, VS Code, etc.) to open data.js

Step 2: Add Your Error Code

Add entries to the errorDatabase object:

'E123': { category: 'Error', title: 'Error Title Here', description: 'What this error means', remedy: 'Step 1: Do this\nStep 2: Do that\nStep 3: Verify' }

Step 3: Save and Reload

Save the file, then refresh the app in your browser to see the new codes.

Support & Troubleshooting

The app won't open

  • Make sure all files (index.html, styles.css, app.js, data.js) are in the same folder
  • Try opening with a different browser
  • Check that there are no file extension issues (e.g., index.html.txt)

Search isn't working

  • Check the browser's Developer Console (F12) for errors
  • Verify data.js has valid JavaScript syntax
  • Try searching with a code that's in the pre-loaded database (E001, A010, etc.)
